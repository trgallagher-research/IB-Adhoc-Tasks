# MM Prototype Sketches

Prototype sketches for the MM competency assessment framework.

## Running the site locally

```bash
cd C:/Users/trgal/projects/IB-Adhoc-Tasks/docs && python -m http.server 8000
```

Open `http://localhost:8000` in your browser. Press `Ctrl+C` in the terminal to stop.

If port 8000 is in use: `python -m http.server 8080`, then visit `http://localhost:8080`.

## Folders

- **`docs/`** — static site (HTML, CSS, JS).
- **`prototype-descriptions.md`** — full text descriptions of all five prototypes.
